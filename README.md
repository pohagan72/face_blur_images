# AI Agent for Blurring Faces in Images

This AI Agent is uses a convolutional d neural network to detect and blur faces in uploaded images. Users can adjust the level of blurring to showcase how adjustable the feature is.

## Installation

Follow these steps to get the application up and running on your local machine:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/pohagan72/face_blur_images.git
    cd face_blur_images
    ```

2.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Place the test image:**
    Ensure `Group1.jpg` is in the root directory of the project, next to `face_blur.py`. This file is used by the "Test File" button in the UI.
